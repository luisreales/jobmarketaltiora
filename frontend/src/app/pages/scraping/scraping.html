ℹ️ About This Tool

  • • Use this page to manually trigger job scraping from LinkedIn, Upwork, or both
  • • Automatic hourly scraping is currently disabled
  • • Each scraping operation may take several minutes depending on the search criteria
  • • Results are saved to the database and appear in the Jobs search
  • • Authentication credentials must be configured in environment variables
  • • For Upwork scraping: Make sure the scraper service is running with docker-compose up scraper-api
